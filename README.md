# E-Commerce Sales Performance Analysis

## Proje Hakkında
Brezilya'nın en büyük e-ticaret platformu Olist'in 2016-2018 verileri kullanılarak 
satış performansı analizi yapılmıştır.

## Kullanılan Araçlar
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Analizler
- Aylık sipariş ve gelir trendi
- Ödeme yöntemi dağılımı
- Kategori bazlı performans analizi
- Teslimat performansı

## Temel Bulgular
- 2017-2018 arasında güçlü büyüme trendi
- watches_gifts en yüksek ortalama sipariş değerine sahip kategori (201 BRL)
- Siparişlerin %93'ü zamanında teslim edilmiş
